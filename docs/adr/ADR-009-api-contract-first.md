# ADR-009 API Contract First

## Context

API perlu terdokumentasi dan stabil sebelum implementasi bertambah kompleks.

## Decision

OpenAPI menjadi sumber kontrak HTTP dan akan diintegrasikan lebih jauh dengan generator.

## Alternatives

- Code-first API
- Dokumentasi setelah implementasi

## Consequences

Kontrak lebih jelas, tetapi spec harus dijaga tetap sinkron dengan implementasi.

## Status

Accepted
